# 깃이란?
- 소스코드를 효율적으로 관리하기 위한 버전 관리 시스템

## 핵심기능
1. 버전관리
2. 백업
3. 협업


## 리눅스 명령어
|명령어|설명|
|------|---|
|pwd|현재 위치 경로|
|ls|현재 디렉터리의 파일이나 디렉터리 확인|
|ls -la|숨긴 파일과 디렉터리까지 확인|
|cd..|상위 디렉터리|
|cd 디렉터리명| 해당 하위 디렉터리로 이동|
|cd ~| 홈 디렉터리로 돌아가기|
|mkdir 디렉터리명| 디렉터리 만들기|
|rm 디렉터리명| 디렉터리 삭제|
|rm -r 디렉터리명| 하위 디렉터리까지 모두 삭제|
|vim 파일명.확장자| vim으로 파일생성|
|cat 파일명.확장자| 파일내용 확인|
